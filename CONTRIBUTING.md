## Contribution Policy

Contributions are welcome, but pull requests must respect the identity, authorship, license, and purpose of this project.

The following changes will not be accepted:

1. Removing or altering copyright notices.
2. Removing attribution to the original author.
3. Removing donation, funding, or support links.
4. Rebranding the project as someone else’s work.
5. Replacing the main project language or documentation without prior discussion.
6. Large automated translations or bulk formatting changes without approval.
7. Changes that make the project harder to maintain or review.

Translations, improvements, bug fixes, and accessibility enhancements are welcome, but they must be submitted in a transparent and reviewable way.

Before making large structural changes, please open an issue first and explain the motivation.
